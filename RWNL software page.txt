<div id="main-title"><h1>RWNL Applications</h1></div>

<a href="publish/FileConverter/publish.html" target="_blank">
<div id="application-block">
  <h2>Convert RWNL dataset to FILMAN or BDF files (FileConverter)</h2>
  <p>For converting trial-by-trial RWNL datasets to FILMAN or BDF records; for each trial, one
record is created based on given offsets from a chosen Event; trials which overlap data
segments between chosen Events may be excluded (see artifact editor description); for
FILMAN conversions a selected set of GVs may be attached to the records; for BDF
conversions, a single GV is selected to create a new Status channel encoding the GV value;
channel selection, decimation and referencing; program permits saving currently selected
parameters to automate repeated use of program.</p>
</div>
</a>

<a href="publish/ASCtoFMConverter/publish.html" target="_blank">
<div id="application-block">
  <h2>ASC to FILMAN converter (ASCtoFMConverter)</h2>
  <p>Converts non-trial-by-trial, continuous record, experiments in RWNL format to &ldquo;buckets of
records&rdquo; in FILMAN format; each bucket contains data segments from one experimental state,
labelled by chosen group variables; choose decimation factor, record length, referencing and
detrending/offset; program permits saving currently selected parameters to automate repeated
use of program.</p>
</div>
</a>

<a href="publish/DatasetReviewer/publish.html" target="_blank">
<div id="application-block">
  <h2>Display RWNL datasets (DatasetReviewer)</h2>
  <p>Opens RWNL dataset and displays BDF file and Event file data in scrollable format; detects
certain errors in RWNL format; can select channels to display including Status channel, manipulate
display decimation, and investigate clock offsets.</p>
</div>
</a>

<a href="publish/ExtractEventsForEEGLAB/publish.html" target="_blank">
<div id="application-block">
  <h2>Extract Event data from RWNL dataset for import into EEGLAB (ExtractEventsForEEGLAB)</h2>
  <p>Opens RWNL datasets and extracts selected information (Event type, Event timing, and GV values) for importation into EEGLAB using a CSV format.</p>
</div>
</a>

<a ExtractEventsForANSLABtarget="_blank">
<div id="application-block">
  <h2>Extract Event data from RWNL dataset for import into ANSLAB (ExtractEventsForANSLAB)</h2>
  <p>Opens RWNL datasets and extracts selected information (Event type, Event timing, and GV values) for importation into ANSLAB using a M-file format.</p>
</div>
</a>

<a href="publish/PreprocessDataset/publish.html" target="_blank">
<div id="application-block">
  <h2>Preprocess RWNL dataset (PreprocessDataset)</h2>
  <p>General preprocessing of RWNL dataset; provides two levels of decimation, before and after
filtering; channel selection and channel name and type editing; referencing including general
linear referencing; surface Laplacian techniques including surface estimation by spherical
harmonics and general sphere fit based on measure electrode locations; select either general
polyharmonic spline or Nunez technique; creates new RWNL dataset.</p>
</div>
</a>

<a href="publish/ConvertEEG2FM/publish.html" target="_blank">
<div id="application-block">
  <h2>EEGLAB to FILMAN converter (ConvertEEG2FM)</h2>
  <p>Converts EEGLAB .SET file (Export output) to FILMAN format on a trial-by-trial basis, one
FILMAN record for each experimental trial; data in the each record is as processed by EEGLAB
(filtered, referenced, etc.); select group variables to be included; create FILMAN file description;
optionally include electrode positioning information.</p>
</div>
</a>
